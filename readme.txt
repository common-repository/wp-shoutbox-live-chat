== Installation ==

1. Upload `Shoutbox_Live_Chat_by_Steveas.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[shoutbox height="400" room="default" userlist="0" userlist_position="left" smilies="1" send_button="1" loggedin_visible="1" guests_visible="1" avatars="1" counter="1"]` in your templates
